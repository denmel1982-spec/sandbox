# Веб-платформа для поиска электронных компонентов

## Описание проекта

Платформа объединяет поставщиков и покупателей электронных компонентов, предоставляя:
- Поиск по каталогу компонентов
- Доску объявлений о покупке/продаже
- Встроенный чат для общения между пользователями
- Управление прайс-листами
- Административную панель для модерации

## Роли пользователей

1. **Гость** – просмотр каталога и доски объявлений
2. **Авторизованный пользователь** – управление объявлениями, прайс-листами, чатами
3. **Администратор** – модерация контента, управление пользователями

## Технологический стек

- **Бэкенд**: PHP 8.3 + Laravel 11
- **Фронтенд**: React
- **База данных**: PostgreSQL 17
- **Кэш и очереди**: Redis
- **Веб-сервер**: Nginx
- **Чат (realtime)**: Centrifugo
- **Окружение**: Docker

## Быстрый старт

### Требования
- Docker и Docker Compose
- Node.js 20+
- PHP 8.3+

### Установка

1. Клонировать репозиторий
2. Запустить контейнеры:
   ```bash
   docker compose up -d
   ```

3. Установить зависимости бэкенда:
   ```bash
   cd backend
   composer install
   cp .env.example .env
   php artisan key:generate
   php artisan migrate --seed
   ```

4. Установить зависимости фронтенда:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## Переменные окружения

См. файлы `.env.example` в папках `backend` и `frontend`.
